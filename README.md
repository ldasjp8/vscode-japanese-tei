# vscode-japanese-tei

TEI/XMLファイルの編集/表示用のVSCode拡張機能

## Features

- リアルタイムプレビュー機能
- `<persName>`タグなどの指定したタグに応じて、プレビュー画面のスタイルを変更する。
- `<ruby>`タグなどに対する入力補助機能を提供する。
